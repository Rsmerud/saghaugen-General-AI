# Saghaugen Uteområdeplan

## Oversikt

Planlegging av uteområdet på Saghaugen - 7 mål tomt (+ 70 mål skog).

**Adresse:** Trondsbuvegen 272, 2110 SLÅSTAD
**Gnr/Bnr:** 26/24/0

## Tomteoversikt (fra kart)

| Sone | Areal ca. | Beskrivelse |
|------|-----------|-------------|
| Rosa (tun) | 1.8 mål | Gårdsplass, innkjørsel, mellom bygninger |
| Gul | ~2.3+ mål | Åpent område øst/nord |
| Grønn | ~3.4 mål | Område vest/sør |
| **Totalt** | **~7 mål** | Hele tomta |

## Eksisterende bygninger

1. **Hovedhus** (nord) - Tømmerhus fra 1943, ~70-80 kvm
2. **Låve** (sør) - 130 kvm totalt, bygget ca. 2000
   - 30 kvm isolert verksted
   - 100 kvm uisolert
